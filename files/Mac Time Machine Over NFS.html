Setting Up OSX Time Machine Over NFS

The issue addressed by this document is that you need a backup device for your Mac that is not an external device or any other type but an NFS device. In other words, you want to use a particular folder in a Linux server as a backup location for your Mac files. This is made possible by using an NFS service available only on Linux or any Unix-like platforms.

Make sure that there is no username in the NFS server that matches that in your Mac.

This guide covers Debian-based server (version 8 [jessie]) and Mac (version 10.12.6).

On the Linux server:

  1. There must be NFS service installed. Otherwise, do this: 
    sudo apt-get install nfs-kernel-server
  2. Make sure that ports used by NFS are allowed by the server's firewall.
  3. Create a user account similar to that on your Mac: 
    sudo useradd -m -u 501 <mac-username>
  4. Log in as the user just created. Create a directory where backup of your Mac will be placed:
    mkdir <backup folder name>
  5. Log out as the user just created.
  6. Add this in the file /etc/exports:
    /home/mac-username/
